TP_1 - Um CRUD incompleto (sem função de deletar) de um sistema de Olímpiadas, sendo possivel adicionar diferentes países, atletas e treinadores de cada país e suas medalhas nos determinados esportes.
compilar e executar:
g++ Atleta.cpp Medalha.cpp Modalidade.cpp Pais.cpp Pessoa.cpp Treinador.cpp Main.cpp -o Tp1 -Wall
./Tp1




TP_2 - Uma releitura do jogo Wordle, feita na biblioteca gráfica Swing do Java, também com a implementação de diferentes perfis, com recordes pessoais e com uma palavra diferente em cada partida, sendo possível até salvar os jogos.
compilar e executar:
javac *.java
java Main
